Session store contract (from D-10, D-12, RESEARCH.md Pattern 5):
  MAX_MESSAGES = 20  (10 turn pairs; D-12 says retain last 10-15 pairs)

  get_or_create_session(session_id: str) -> dict:
    returns {"id": session_id, "turns": list[dict]}
    "turns" is the mutable list stored in _sessions[session_id]
    creates an empty list if session_id not in _sessions

  append_turn(session_id: str, role: str, content: str) -> None:
    appends {"role": role, "content": content} to _sessions[session_id]
    applies sliding window: if len(turns) > MAX_MESSAGES, del turns[:2]
    always drops exactly 2 messages (preserves role alternation)

Thread safety: threading.Lock() guards all reads and writes to _sessions dict.

From RESEARCH.md Pattern 5 — exact module-level state shape:
  _sessions: dict[str, list[dict]] = {}
  _lock = threading.Lock()
